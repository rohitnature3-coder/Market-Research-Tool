# SCREEN BY SCREEN VIBE PROTOTYPE

## Overview
This document outlines the comprehensive screen-by-screen vibe prototype of the Market Research Tool. Each screen includes design specifications, color palette, typography, layouts, components, interactions, animations, accessibility guidelines, and design tokens.

---

### 1. Login Screen
- **Design Specifications:**
  - Simple, clean layout with central form input.
- **Color Palette:**
  - Background: Transparent white.
  - Buttons: Soft blue.
- **Typography:**
  - Heading: Bold, 24px.
  - Input Fields: Regular, 16px.
- **Components:**
  - Form inputs, Sign In button, Forgot Password link.
- **Interactions:**
  - Focus states on inputs, hover effects on buttons.
- **Animations:**
  - Fade-in effect for screen elements.

---

## Design Patterns
- **Glass Morphism:**
  - Apply blurred background effects with subtle shadows across all screens to create depth and focus on main content.

## Responsive Breakpoints
- **Mobile:** 375px;
- **Tablet:** 768px;
- **Desktop:** 1024px or more.

## Micro-Interactions
- **On Button Clicks:**
  - Subtle enlargements or color changes for feedback.
- **Hover Effects:**
  - Elements may slightly lift, indicating interactivity.

## Accessibility Guidelines
- Ensure color contrast meets WCAG standards.
- All interactive elements should have text labels and focus states.

## Design Tokens
- **Color Tokens** for reuse across UI components:
  - Primary Color: `#007BFF`;
  - Secondary Color: `#6C757D`;
- **Typography Tokens** for fonts and sizes.
