## User Flow
1. User lands on Home Page
2. User searches for studies using the search bar
3. User selects a study from the results
4. User views study detail and can save it
5. User navigates to their profile to review saved studies

## Design Mood/Vibe Specifications
- **Color Palette**: Blue, Green, and White
- **Font Style**: Modern Sans-serif fonts for readability
- **Imagery**: Use professional, high-quality images related to market research.
- **Overall Feel**: Clean, organized, and user-friendly.
